# QuantBot
基于强化学习的自动交易 LLM Agent

### Setup Virtual Environment
For Win
```powershell
.\setup\setup_env.bat
```

For Linux
```bash
./setup/setup_env.sh
```
### Test
Download LLM (GPU Required)
```bash
python ./quantbot/workflow/download_llm.py
```
Test Simulator
```bash
python ./quantbot/test/test_simulator.py
```
